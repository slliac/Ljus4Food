\subsection{Detailed Description}
Constraint that asserts that the string it is evaluated for matches a regular expression.

Checks a given value using the Perl Compatible Regular Expression extension in P\+HP. The pattern is matched by executing preg\+\_\+match().

The pattern string passed in the constructor.
