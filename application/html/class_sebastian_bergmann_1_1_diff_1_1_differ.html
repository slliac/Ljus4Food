Returns the diff between two arrays or strings as array.

Each array element contains two elements:

  • [0] => mixed $token
  • [1] => 2|1|0

2: REMOVED: $token was removed from $from

  • 1: ADDED: $token was added to $from
  • 0: OLD: $token is not changed in $to
